21    /**
22    * A simple implementation of the {@link JsonElementVisitor} that simply delegates the method
23    * invocation onto a {@code delegate} instance of the {@link JsonElementVisitor}. This object
24    * can be used to build a chain of visitors such that each Visitor instance can perform some
25    * operation on the {@link JsonElement} and then pass on the input to the delegate. This kind
26    * of pattern is sometimes referred as a "Chain of Responsibility".
27    *
28    * <p>The following is an example use case:
29    *
30    * <pre>
31    * class JsonEscapingVisitor extends DelegatingJsonElementVisitor {
32    * public JsonEscapingVisitor(JsonElementVisitor) {
33    * super(visitor);
34    * }
35    *
36    * public void visitPrimitive(JsonPrimitive primitive) {
37    * JsonPrimitive escapedPrimitive = escapePrimitiveObject(primitive);
38    * super.visitPrimitive(escapedPrimitive);
39    * }
40    * }
41    *
42    * JsonElementVisitor visitor = new JsonEscapingVisitor(new FormattingVisitor());
43    * </pre></p>
44    *
45    * @author Joel Leitch
46    */
