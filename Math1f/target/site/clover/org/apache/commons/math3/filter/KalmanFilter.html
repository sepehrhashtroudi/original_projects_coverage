33    /**
34    * Implementation of a Kalman filter to estimate the state <i>x<sub>k</sub></i>
35    * of a discrete-time controlled process that is governed by the linear
36    * stochastic difference equation:
37    *
38    * <pre>
39    * <i>x<sub>k</sub></i> = <b>A</b><i>x<sub>k-1</sub></i> + <b>B</b><i>u<sub>k-1</sub></i> + <i>w<sub>k-1</sub></i>
40    * </pre>
41    *
42    * with a measurement <i>x<sub>k</sub></i> that is
43    *
44    * <pre>
45    * <i>z<sub>k</sub></i> = <b>H</b><i>x<sub>k</sub></i> + <i>v<sub>k</sub></i>.
46    * </pre>
47    *
48    * <p>
49    * The random variables <i>w<sub>k</sub></i> and <i>v<sub>k</sub></i> represent
50    * the process and measurement noise and are assumed to be independent of each
51    * other and distributed with normal probability (white noise).
52    * <p>
53    * The Kalman filter cycle involves the following steps:
54    * <ol>
55    * <li>predict: project the current state estimate ahead in time</li>
56    * <li>correct: adjust the projected estimate by an actual measurement</li>
57    * </ol>
58    * <p>
59    * The Kalman filter is initialized with a {@link ProcessModel} and a
60    * {@link MeasurementModel}, which contain the corresponding transformation and
61    * noise covariance matrices. The parameter names used in the respective models
62    * correspond to the following names commonly used in the mathematical
63    * literature:
64    * <ul>
65    * <li>A - state transition matrix</li>
66    * <li>B - control input matrix</li>
67    * <li>H - measurement matrix</li>
68    * <li>Q - process noise covariance matrix</li>
69    * <li>R - measurement noise covariance matrix</li>
70    * <li>P - error covariance matrix</li>
71    * </ul>
72    *
73    * @see <a href="http://www.cs.unc.edu/~welch/kalman/">Kalman filter
74    * resources</a>
75    * @see <a href="http://www.cs.unc.edu/~welch/media/pdf/kalman_intro.pdf">An
76    * introduction to the Kalman filter by Greg Welch and Gary Bishop</a>
77    * @see <a href="http://academic.csuohio.edu/simond/courses/eec644/kalman.pdf">
78    * Kalman filter example by Dan Simon</a>
79    * @see ProcessModel
80    * @see MeasurementModel
81    * @since 3.0
82    * @version $Id$
83    */
