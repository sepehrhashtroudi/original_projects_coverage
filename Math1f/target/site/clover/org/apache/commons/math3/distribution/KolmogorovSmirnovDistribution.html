35    /**
36    * Implementation of the Kolmogorov-Smirnov distribution.
37    *
38    * <p>
39    * Treats the distribution of the two-sided {@code P(D_n < d)} where
40    * {@code D_n = sup_x |G(x) - G_n (x)|} for the theoretical cdf {@code G} and
41    * the empirical cdf {@code G_n}.
42    * </p>
43    * <p>
44    * This implementation is based on [1] with certain quick decisions for extreme
45    * values given in [2].
46    * </p>
47    * <p>
48    * In short, when wanting to evaluate {@code P(D_n < d)}, the method in [1] is
49    * to write {@code d = (k - h) / n} for positive integer {@code k} and
50    * {@code 0 <= h < 1}. Then {@code P(D_n < d) = (n! / n^n) * t_kk}, where
51    * {@code t_kk} is the {@code (k, k)}'th entry in the special matrix
52    * {@code H^n}, i.e. {@code H} to the {@code n}'th power.
53    * </p>
54    * <p>
55    * References:
56    * <ul>
57    * <li>[1] <a href="http://www.jstatsoft.org/v08/i18/">
58    * Evaluating Kolmogorov's Distribution</a> by George Marsaglia, Wai
59    * Wan Tsang, and Jingbo Wang</li>
60    * <li>[2] <a href="http://www.jstatsoft.org/v39/i11/">
61    * Computing the Two-Sided Kolmogorov-Smirnov Distribution</a> by Richard Simard
62    * and Pierre L'Ecuyer</li>
63    * </ul>
64    * Note that [1] contains an error in computing h, refer to
65    * <a href="https://issues.apache.org/jira/browse/MATH-437">MATH-437</a> for details.
66    * </p>
67    *
68    * @see <a href="http://en.wikipedia.org/wiki/Kolmogorov-Smirnov_test">
69    * Kolmogorov-Smirnov test (Wikipedia)</a>
70    * @version $Id$
71    */
