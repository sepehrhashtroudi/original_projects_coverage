33    /**
34    * Represents a polynomial spline function.
35    * <p>
36    * A <strong>polynomial spline function</strong> consists of a set of
37    * <i>interpolating polynomials</i> and an ascending array of domain
38    * <i>knot points</i>, determining the intervals over which the spline function
39    * is defined by the constituent polynomials. The polynomials are assumed to
40    * have been computed to match the values of another function at the knot
41    * points. The value consistency constraints are not currently enforced by
42    * <code>PolynomialSplineFunction</code> itself, but are assumed to hold among
43    * the polynomials and knot points passed to the constructor.</p>
44    * <p>
45    * N.B.: The polynomials in the <code>polynomials</code> property must be
46    * centered on the knot points to compute the spline function values.
47    * See below.</p>
48    * <p>
49    * The domain of the polynomial spline function is
50    * <code>[smallest knot, largest knot]</code>. Attempts to evaluate the
51    * function at values outside of this range generate IllegalArgumentExceptions.
52    * </p>
53    * <p>
54    * The value of the polynomial spline function for an argument <code>x</code>
55    * is computed as follows:
56    * <ol>
57    * <li>The knot array is searched to find the segment to which <code>x</code>
58    * belongs. If <code>x</code> is less than the smallest knot point or greater
59    * than the largest one, an <code>IllegalArgumentException</code>
60    * is thrown.</li>
61    * <li> Let <code>j</code> be the index of the largest knot point that is less
62    * than or equal to <code>x</code>. The value returned is <br>
63    * <code>polynomials[j](x - knot[j])</code></li></ol></p>
64    *
65    * @version $Id$
66    */
