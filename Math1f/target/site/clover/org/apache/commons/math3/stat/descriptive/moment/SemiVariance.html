27    /**
28    * <p>Computes the semivariance of a set of values with respect to a given cutoff value.
29    * We define the <i>downside semivariance</i> of a set of values <code>x</code>
30    * against the <i>cutoff value</i> <code>cutoff</code> to be <br/>
31    * <code>&Sigma; (x[i] - target)<sup>2</sup> / df</code> <br/>
32    * where the sum is taken over all <code>i</code> such that <code>x[i] < cutoff</code>
33    * and <code>df</code> is the length of <code>x</code> (non-bias-corrected) or
34    * one less than this number (bias corrected). The <i>upside semivariance</i>
35    * is defined similarly, with the sum taken over values of <code>x</code> that
36    * exceed the cutoff value.</p>
37    *
38    * <p>The cutoff value defaults to the mean, bias correction defaults to <code>true</code>
39    * and the "variance direction" (upside or downside) defaults to downside. The variance direction
40    * and bias correction may be set using property setters or their values can provided as
41    * parameters to {@link #evaluate(double[], double, Direction, boolean, int, int)}.</p>
42    *
43    * <p>If the input array is null, <code>evaluate</code> methods throw
44    * <code>IllegalArgumentException.</code> If the array has length 1, <code>0</code>
45    * is returned, regardless of the value of the <code>cutoff.</code>
46    *
47    * <p><strong>Note that this class is not intended to be threadsafe.</strong> If
48    * multiple threads access an instance of this class concurrently, and one or
49    * more of these threads invoke property setters, external synchronization must
50    * be provided to ensure correct results.</p>
51    *
52    * @since 2.1
53    * @version $Id$
54    */
