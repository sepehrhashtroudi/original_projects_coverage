82    /**
83    * This class guesses harmonic coefficients from a sample.
84    * <p>The algorithm used to guess the coefficients is as follows:</p>
85    *
86    * <p>We know f (t) at some sampling points t<sub>i</sub> and want to find a,
87    * &omega; and &phi; such that f (t) = a cos (&omega; t + &phi;).
88    * </p>
89    *
90    * <p>From the analytical expression, we can compute two primitives :
91    * <pre>
92    * If2 (t) = &int; f<sup>2</sup> = a<sup>2</sup> &times; [t + S (t)] / 2
93    * If'2 (t) = &int; f'<sup>2</sup> = a<sup>2</sup> &omega;<sup>2</sup> &times; [t - S (t)] / 2
94    * where S (t) = sin (2 (&omega; t + &phi;)) / (2 &omega;)
95    * </pre>
96    * </p>
97    *
98    * <p>We can remove S between these expressions :
99    * <pre>
100    * If'2 (t) = a<sup>2</sup> &omega;<sup>2</sup> t - &omega;<sup>2</sup> If2 (t)
101    * </pre>
102    * </p>
103    *
104    * <p>The preceding expression shows that If'2 (t) is a linear
105    * combination of both t and If2 (t): If'2 (t) = A &times; t + B &times; If2 (t)
106    * </p>
107    *
108    * <p>From the primitive, we can deduce the same form for definite
109    * integrals between t<sub>1</sub> and t<sub>i</sub> for each t<sub>i</sub> :
110    * <pre>
111    * If2 (t<sub>i</sub>) - If2 (t<sub>1</sub>) = A &times; (t<sub>i</sub> - t<sub>1</sub>) + B &times; (If2 (t<sub>i</sub>) - If2 (t<sub>1</sub>))
112    * </pre>
113    * </p>
114    *
115    * <p>We can find the coefficients A and B that best fit the sample
116    * to this linear expression by computing the definite integrals for
117    * each sample points.
118    * </p>
119    *
120    * <p>For a bilinear expression z (x<sub>i</sub>, y<sub>i</sub>) = A &times; x<sub>i</sub> + B &times; y<sub>i</sub>, the
121    * coefficients A and B that minimize a least square criterion
122    * &sum; (z<sub>i</sub> - z (x<sub>i</sub>, y<sub>i</sub>))<sup>2</sup> are given by these expressions:</p>
123    * <pre>
124    *
125    * &sum;y<sub>i</sub>y<sub>i</sub> &sum;x<sub>i</sub>z<sub>i</sub> - &sum;x<sub>i</sub>y<sub>i</sub> &sum;y<sub>i</sub>z<sub>i</sub>
126    * A = ------------------------
127    * &sum;x<sub>i</sub>x<sub>i</sub> &sum;y<sub>i</sub>y<sub>i</sub> - &sum;x<sub>i</sub>y<sub>i</sub> &sum;x<sub>i</sub>y<sub>i</sub>
128    *
129    * &sum;x<sub>i</sub>x<sub>i</sub> &sum;y<sub>i</sub>z<sub>i</sub> - &sum;x<sub>i</sub>y<sub>i</sub> &sum;x<sub>i</sub>z<sub>i</sub>
130    * B = ------------------------
131    * &sum;x<sub>i</sub>x<sub>i</sub> &sum;y<sub>i</sub>y<sub>i</sub> - &sum;x<sub>i</sub>y<sub>i</sub> &sum;x<sub>i</sub>y<sub>i</sub>
132    * </pre>
133    * </p>
134    *
135    *
136    * <p>In fact, we can assume both a and &omega; are positive and
137    * compute them directly, knowing that A = a<sup>2</sup> &omega;<sup>2</sup> and that
138    * B = - &omega;<sup>2</sup>. The complete algorithm is therefore:</p>
139    * <pre>
140    *
141    * for each t<sub>i</sub> from t<sub>1</sub> to t<sub>n-1</sub>, compute:
142    * f (t<sub>i</sub>)
143    * f' (t<sub>i</sub>) = (f (t<sub>i+1</sub>) - f(t<sub>i-1</sub>)) / (t<sub>i+1</sub> - t<sub>i-1</sub>)
144    * x<sub>i</sub> = t<sub>i</sub> - t<sub>1</sub>
145    * y<sub>i</sub> = &int; f<sup>2</sup> from t<sub>1</sub> to t<sub>i</sub>
146    * z<sub>i</sub> = &int; f'<sup>2</sup> from t<sub>1</sub> to t<sub>i</sub>
147    * update the sums &sum;x<sub>i</sub>x<sub>i</sub>, &sum;y<sub>i</sub>y<sub>i</sub>, &sum;x<sub>i</sub>y<sub>i</sub>, &sum;x<sub>i</sub>z<sub>i</sub> and &sum;y<sub>i</sub>z<sub>i</sub>
148    * end for
149    *
150    * |--------------------------
151    * \ | &sum;y<sub>i</sub>y<sub>i</sub> &sum;x<sub>i</sub>z<sub>i</sub> - &sum;x<sub>i</sub>y<sub>i</sub> &sum;y<sub>i</sub>z<sub>i</sub>
152    * a = \ | ------------------------
153    * \| &sum;x<sub>i</sub>y<sub>i</sub> &sum;x<sub>i</sub>z<sub>i</sub> - &sum;x<sub>i</sub>x<sub>i</sub> &sum;y<sub>i</sub>z<sub>i</sub>
154    *
155    *
156    * |--------------------------
157    * \ | &sum;x<sub>i</sub>y<sub>i</sub> &sum;x<sub>i</sub>z<sub>i</sub> - &sum;x<sub>i</sub>x<sub>i</sub> &sum;y<sub>i</sub>z<sub>i</sub>
158    * &omega; = \ | ------------------------
159    * \| &sum;x<sub>i</sub>x<sub>i</sub> &sum;y<sub>i</sub>y<sub>i</sub> - &sum;x<sub>i</sub>y<sub>i</sub> &sum;x<sub>i</sub>y<sub>i</sub>
160    *
161    * </pre>
162    * </p>
163    *
164    * <p>Once we know &omega;, we can compute:
165    * <pre>
166    * fc = &omega; f (t) cos (&omega; t) - f' (t) sin (&omega; t)
167    * fs = &omega; f (t) sin (&omega; t) + f' (t) cos (&omega; t)
168    * </pre>
169    * </p>
170    *
171    * <p>It appears that <code>fc = a &omega; cos (&phi;)</code> and
172    * <code>fs = -a &omega; sin (&phi;)</code>, so we can use these
173    * expressions to compute &phi;. The best estimate over the sample is
174    * given by averaging these expressions.
175    * </p>
176    *
177    * <p>Since integrals and means are involved in the preceding
178    * estimations, these operations run in O(n) time, where n is the
179    * number of measurements.</p>
180    */
