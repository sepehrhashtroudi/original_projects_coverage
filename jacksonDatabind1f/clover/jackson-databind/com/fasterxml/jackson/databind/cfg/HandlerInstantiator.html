11    /**
12    * Helper class used for handling details of creating handler instances (things
13    * like {@link JsonSerializer}s, {@link JsonDeserializer}s, various type
14    * handlers) of specific types. Actual handler type has been resolved at this
15    * point, so instantiator is strictly responsible for providing a configured
16    * instance by constructing and configuring a new instance, or possibly by
17    * recycling a shared instance. One use case is that of allowing
18    * dependency injection, which would otherwise be difficult to do.
19    *<p>
20    * Custom instances are allowed to return null to indicate that caller should
21    * use the default instantiation handling (which just means calling no-argument
22    * constructor via reflection).
23    *<p>
24    * Care has to be taken to ensure that if instance returned is shared, it will
25    * be thread-safe; caller will not synchronize access to returned instances.
26    */
