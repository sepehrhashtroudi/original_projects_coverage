310    * MIN_OCCURRENCES_REQUIRED_TO_ALIAS_LITERAL times. Aliasing a keyword has a
311    * cost (e.g. 'var XX=true;' costs 12 bytes). We make up for this
312    * cost by replacing references to the keyword with variables that
313    * have shorter names. If there are only a few references to a
314    * keyword, the cost outweighs the benefit. It is not possible to
315    * determine the exact break-even point without compiling twice
316    * (once with aliasing, another without) and comparing the
317    * post-gzipped size, so we define a minimum number of references
318    * required in order to alias. We choose 6 because the alias cost is
319    * ~7-12 bytes (12 bytes for 'var XX=true;', 7 bytes for a
320    * subsequent declaration that does not require its own 'var ' or
321    * semicolon, e.g. var XX=true,XY=null;), but each reference saves
