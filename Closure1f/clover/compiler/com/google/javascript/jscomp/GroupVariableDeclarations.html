27    /**
28    * Groups multiple variable declarations (that may or may not be contiguous)
29    * in the same scope into a single one. i.e.
30    *
31    * <pre>
32    * var a, b = 10;
33    * f1();
34    * var c, d;
35    * ... some other code ...
36    * var x, y, z = 100;
37    * ... some other code ...
38    * var p = 200, q = 300;
39    * </pre>
40    *
41    * becomes:
42    *
43    * <pre>
44    * var a, b = 10, c, d, x, y, z;
45    * f1();
46    * ... some other code ...
47    * z = 100;
48    * ... some other code ...
49    * var p = 200, q = 300;
50    * </pre>
51    *
52    * This reduces the generated uncompressed code size.
53    *
54    * For any scope, we use the first VAR statement as the statement to collapse
55    * the other declarations into. For other VAR statements in the scope, we only
56    * consider ones that either (a) have no variable that is initialized in the
57    * the statement, or (b) have exactly one variable that is initialized (e.g.
58    * the "var x, y, z = 100;" statement in the example above. VAR statements
59    * with more than one variable initialization are not collapsed. This is
60    * because doing so would increase uncompressed code size.
61    *
62    */
