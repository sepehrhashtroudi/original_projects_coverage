40    // This algorithm works in two stages. First, the depth-first search (DFS)
41    // tree is calculated with A as the root. During when constructing the DFS
42    // tree, back edges are recorded. A back edge is a non-tree edge (X -> Y)
43    // where X is an descendant of Y in the DFS tree. The second step does a
44    // recursive traversal of the graph. Back edges are ignored during the
45    // recursive traversal, thus no cycles are encountered. Any recursive branch

47    // from the entry node to the exit without any nodes that satisfy the
