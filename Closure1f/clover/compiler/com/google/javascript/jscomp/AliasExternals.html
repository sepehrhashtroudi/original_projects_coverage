38    /**
39    * <p>AliasExternals provides wrappers and aliases for external globals and
40    * properties to that they can be referenced by their full name only once
41    * instead of in all use sites.</p>
42    *
43    * <p>The property alias pass creates function wrappers for properties that need
44    * to be accessed externally. These function wrappers are then used by all
45    * internal calls to the property, and the names will be compressed during the
46    * RenamePrototypes step.</p>
47    *
48    * <p>Properties that are accessed externally are either system functions
49    * (i.e. window.document), or used by JavaScript embedded on a page.</p>
50    *
51    * <p>Properties that are r-values are changed to use array notation with
52    * a string that has been defined separately and can be compressed
53    * i.e. document.window -> document[PROP_window].</p>
54    *
55    * <p>Properties that are l-values and can be renamed are renamed to
56    * SETPROP_prop. I.e. node.innerHTML = '&lt;div&gt;Hello&lt;/div&gt;' ->
57    * SETPROP_innerHTML(node, '&lt;div&gt;hello&lt;/div&gt;').</p>
58    *
59    * <p>Properties will only be renamed if they are used more than requiredUsage_
60    * times, as there is overhead for adding the accessor and mutator functions.
61    * This is initialized to DEFAULT_REQUIRED_USAGE (=4), but can be
62    * overridden.</p>
63    *
64    * <p>Certain usages (increment, decrement) won't be addressed, as they would
65    * require a getprop, setprop, and custom logic, and aren't worth
66    * optimizing.</p>
67    *
68    * <p>The global alias pass creates aliases for global variables and functions
69    * that are declared or need to be used externally. These aliases are then used
70    * throughout the code, and will be compressed during the RenameVars step.</p>
71    *
72    * <p>Globals are aliased by inserting code like "var GLOBAL_window = window;"
73    * and then replacing all other uses of "window" with "GLOBAL_window."</p>
74    *
75    * <p>Globals that are l-values are not aliased.</p>
76    *
77    */
